# 🧈 奶油分帳 Butter Split

韓式奶油風的多人分帳網站。跟朋友出去玩、聚餐、旅行，把帳記在這裡，最後一鍵算清。

## 功能

- 🗂️ **多計劃管理**：可建立多個分帳計劃，各自獨立
- 🔒 **結案鎖定**：計劃結案後帳目唯讀，只能查看結算結果
- 🐻 **手動新增成員**：計劃內自由加人
- 💸 **多幣別記帳**：支援 12 種幣別，自動換算成主幣別
- 💱 **共用匯率表**：全站維護一份匯率，可手動輸入或一鍵抓取即時匯率
- ⚖️ **Splitwise 式結算**：算出最少轉帳次數的還款方案
- ✅ **結清紀錄**：每筆還款可標記已結清，可反悔
- 📒 **帳目明細**：點開看每筆帳分給誰、各分擔多少
- ✏️ **帳目編輯**：建立後仍可修改金額、付款人、分攤對象、日期、備註等，不需刪除重建

## 使用方式

直接在瀏覽器打開 `index.html` 即可使用。預設資料儲存在瀏覽器本機（localStorage）。

### Google 試算表同步

1. 建立 [Google 試算表](https://sheets.google.com)，在 Apps Script 貼上 `google-apps-script/Code.gs` 並部署為「網頁應用程式」（任何人可存取）
2. 把部署 URL **寫進** `index.html` 的 `SHEETS_API_URL` 常數
3. 進入計劃後按 **🔗 分享** 傳給朋友——對方**不用任何設定**，打開連結即可同步（約每 4 秒自動刷新）

## 線上版

透過 GitHub Pages 部署後，可直接用網址開啟，分享給朋友。

## 技術

純 HTML / CSS / JavaScript，無框架、無後端、無相依套件。
