# 🌊 海鹽奶油分帳 Butter Split

海鹽奶油風的多人分帳網站。  
跟朋友出去玩、聚餐、旅行，把每一筆花費記在這裡，最後一鍵算清。

整體採用粉藍、奶白、貝殼粉與海洋元素設計，支援手機與電腦瀏覽器使用。

## 功能

- 🧳 **多旅程管理**：可建立多個分帳旅程，各自獨立
- 🌙 **結案鎖定**：旅程結案後帳目唯讀，只能查看結算結果
- 🐬 **手動新增成員**：旅程內可自由新增成員
- 🦀 **多幣別記帳**：支援 12 種幣別，自動換算成主要幣別
- 🪸 **帳目編輯**：建立後可修改名目、金額、幣別、付款人、分攤方式、分攤對象與日期，不需刪除重新建立
- ⚓ **共用匯率表**：全站共用一份匯率，可手動輸入或一鍵抓取即時匯率
- 🌅 **每人餘額**：自動計算每位成員應收或應付金額
- ⛵ **最少轉帳結算**：使用 Splitwise 式算法，算出最少轉帳次數的還款方案
- ✅ **結清紀錄**：每筆還款可標記已結清，也可以取消結清
- 🐠 **帳目明細**：可展開查看付款人、分攤明細與匯率資訊
- ☁️ **Google Sheets 雲端同步**：多人可透過分享連結即時同步資料
- 🪼 **分享連結**：建立旅程後可分享網址，朋友開啟即可共同記帳
- 🌊 **海鹽奶油風介面**：粉藍、奶白、貝殼粉與海洋圖示設計

## 使用方式

直接在瀏覽器打開 `index.html` 即可使用。

若沒有設定 Google Sheets 雲端同步，資料會預設儲存在瀏覽器本機的 `localStorage`。

### Google 試算表同步

1. 建立 [Google 試算表](https://sheets.google.com)
2. 開啟 Apps Script
3. 貼上 `Code.gs`
4. 部署為「網頁應用程式」
5. 存取權限設定為「任何人」
6. 將部署網址貼到 `index.html` 裡的 `SHEETS_API_URL`

const SHEETS_API_URL = '你的 Apps Script 網頁應用程式網址';
